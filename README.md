# NextGen-Community-Hub

NextGen Community Hub is a community resource website designed to help residents (particularly teenagers) in Monmouth County, NJ, easily find local organizations, services, programs, and events. The platform serves as a centralized hub where users can explore opportunities and support resources available in their community.

## Features
- **Interactive Resource Directory** with search and filtering  
- **Spotlight Section** highlighting featured community organizations  
- **Resource Submission Form** for users to suggest new resources  
- Listings for programs, events, and opportunities  
- **Responsive design** for accessibility across devices  

## Purpose
The goal of this project is to make community resources easier to discover and connect Monmouth County residents (especially teens) with organizations that provide support, opportunities, and services.
